# About

SharpEngine is a purely .NET game engine that is designed for users that want a traditional .NET development experience.

This repository contains the libraries commonly used across the SharpEngine components.

### Prerequisits

https://pnpm.io/
https://nodejs.org/en
https://dotnet.microsoft.com/en-us/download/dotnet/10.0
